====== About

This tiny module adds new filter to views, that can help you
get products in stock. It check's stock control activity:
product will be shown if stock control inactive or active
and stock value more than 0.
Typical use as exposed filter for customers.


====== Installation

After module activation you can see new filter in "Stock" group.




====== Описание

Этот маленький модуль добавляет фильтр для Views, с помощью которого
можно выбирать товары, имеющиеся на складе. Проверяется активность контроля
складского остатка: товар выводится если он неактивен или если активен
и количество остатка больше 0.
Обычно используется как раскрытый фильтр для покупателей.


====== Установка

После активации модуля вы увидите новый фильтр в групп "Склад".
